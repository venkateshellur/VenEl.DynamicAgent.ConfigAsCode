# VenEl.DynamicAgents

Welcome to the official documentation for the **VenEl Multi-Agent Config-As-Code Framework**.

## Overview

The VenEl framework allows you to orchestrate intelligent AI agents directly from YAML configuration files. It completely separates your business logic (agent orchestration) from your application logic.

### Key Concepts

- **Agents**: Define your AI personas, tools, and system prompts in `agents.yaml`.
- **Workflows**: String your agents together into execution pipelines in `workflows.yaml`.
- **Engine**: The `SequentialWorkflowEngine` executes your pipelines using dependency injection.
